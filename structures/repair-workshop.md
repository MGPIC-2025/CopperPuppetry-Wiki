# 维修工坊

## 📋 建筑信息

| 属性 | 详情 |
|------|------|
| **类型** | 辅助/攻击建筑 |
| **建造成本** | 8 精炼铜锭 + 3 召回齿轮 + 3 心源矿尘 |
| **作用范围** | 3格 |

**描述**：一个带有可开合顶棚的圆形工坊，内部有多个机械臂。友方铜偶经过时，机械臂会自动为其进行修复和保养，发出叮叮当当的敲打声。

## ⚡ 建筑效果

### 基础属性
- **生命值**：200
- **攻击力**：25
- **攻击范围**：3格

### 特殊能力
**自动维修**：为经过的友方铜偶进行修复

## 🎯 战术定位

**维修支援塔** - 持续修复周围建筑和英雄。

### 优势
- 持续恢复能力
- 提升队伍续航
- 配合坦克持久战

### 劣势
- 无攻击力
- 成本较高
- 需要前期投入

## 💡 布局建议

- 放在主战场中心
- 保护重要建筑
- 配合坦克英雄

::: tip 💡 长期战斗
维修车间是打持久战的关键，配合Wesley等自愈坦克，几乎打不死。
:::
