# 充能线圈 <el-icon><Warning /></el-icon>

## <el-icon><Document /></el-icon> 建筑信息

| 属性 | 详情 |
|------|------|
| **类型** | <el-icon><Warning /></el-icon> **敌方建筑**（怪物生成器） |
| **是否可建造** | <el-icon><Warning /></el-icon> **否** - 这是敌方建筑 |
| **位置** | 房间中心（4,4位置） |

**描述**：一个由巨大铜线圈缠绕而成的塔状建筑，线圈中央有电弧在不断跳跃。

## <el-icon><Warning /></el-icon> 危险功能

### 基础属性
- **生命值**：200
- **攻击能力**：无
- **攻击范围**：0
- **所属**：敌方（owned=false）

### <el-icon><Warning /></el-icon> 怪物生成机制
**持续刷怪**：
- 每回合有 **20% 概率**生成敌人
- 只要充能线圈存在，就会持续刷怪
- 生成的敌人等级与房间等级相同
- 生成位置为房间内的空白位置

**代码实现**：
```moonbit
// 游戏回合结束时如果还有充能线圈，则有概率持续生成敌人
fn summon_enemy_while_game_round_pass() -> Unit {
  if check_probability(0.2) {  // 20%概率
    // 检查房间中心是否有充能线圈
    if structure.structure_base.name == "充能线圈" {
      // 生成敌人
      let enemy = summon_enemy(level)
      // 在空位置放置敌人
    }
  }
}
```

## <el-icon><Aim /></el-icon> 战术对策

### <el-icon><Warning /></el-icon> 必须优先摧毁！

**为什么必须优先清除**：
- 持续刷怪会让敌人数量失控
- 拖延时间只会让战况更加恶化
- 清除后房间才能安全探索

### 推荐战术

1. **快速集火**
   - 进入房间后立即锁定充能线圈
   - 使用高攻击力英雄集火
   - 优先级高于普通敌人

2. **推荐英雄**
   - 镜光折射塔（高伤害）
   - 咝咝（高爆发输出）
   - 公输（坚固建筑，可快速拆除）

3. **时间就是金钱**
   - 越快清除越好
   - 每多等一回合，就多20%刷怪风险
   - 建议3回合内清除

## 💡 识别方法

- **位置特征**：永远在房间中心
- **外观特征**：大型线圈，电弧跳跃
- **标记**：地图上会显示为敌方建筑
