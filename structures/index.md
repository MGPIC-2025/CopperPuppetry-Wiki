# 建筑百科

建筑系统是铜偶戏的核心机制之一。这些建筑都是铜曜王朝的科技遗产！

## 🏗️ 建筑系统概述

在铜曜王朝时代，这些建筑遍布整个王城：攻击塔守卫城墙，矿钻采集资源，工坊维护设施。大休眠后，工匠职业的铜偶们可以重建这些古老的建筑。

### 建筑分类
- ⚔️ **攻击建筑**：自动攻击敌人（4种）
- 💰 **资源建筑**：生产精炼铜锭（1种）
- 🛡️ **防御建筑**：阻挡敌人前进（1种）
- ✨ **辅助建筑**：提供增益效果（2种）

游戏中共有 **8 种建筑**。

## 💡 建筑说明

所有建筑都是铜曜王朝的技术遗产，只有工匠职业的铜偶可以建造它们。

### 建筑特点
- **攻击建筑**：齿轮箭塔性价比高，镜光折射塔和风暴铁砧威力强大但成本高
- **资源建筑**：心源矿钻是经济核心，持续产出精炼铜锭
- **防御建筑**：铆接壁垒用于阻挡敌人，构建防线
- **辅助建筑**：充能线圈和共鸣警钟提供增益效果

### 建造要点
- 建造需要消耗精炼铜锭和其他资源
- 需要工匠职业英雄（波波、特斯拉、公输、丁格、小卯）
- 合理布局建筑可以最大化防御效果

## 🎯 建筑使用建议

### 工匠英雄推荐
建造建筑需要工匠职业英雄：
- **小卯** - 速建天才，建造最快
- **公输** - 建筑大师，建筑更坚固
- **特斯拉** - 能量专家，辅助建筑效果更好
- **波波** - 维修专家，建筑恢复更快
- **丁格** - 资源专家，矿钻产出更多

### 推荐装备
- **不稳定的欢乐扳手** - 提升攻击力和闪避
- **万用齿轮口袋** - 提升攻击力和闪避
- **掘掘的幸运罗盘** - 通用装备

### 建造技巧
1. **经济优先**：优先建造心源矿钻，快速发展经济
2. **防御布局**：用铆接壁垒阻挡敌人，齿轮箭塔提供火力
3. **辅助增益**：充能线圈和共鸣警钟提供全队增益
4. **高级建筑**：后期建造镜光折射塔和风暴铁砧
