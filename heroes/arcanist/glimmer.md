# 菲尼 (Glimmer)

## 📋 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 菲尼 |
| **英文名** | Glimmer |
| **职业** | 奥术师 (Arcanist) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## 📖 背景故事

王朝著名剧场"星辉之歌"的首席灯光师。认为魔法不过是另一种形式的舞台光效。她的数据库里存储了上万种灯光配色方案。对她而言，击败敌人的最大成就感来自于技能释放时那一瞬间的色彩搭配是否足够"高级"。

## 📊 基础属性

- **生命值**：50
- **攻击力**：20
- **速度**：3
- **防御力**：5
- **闪避**：10

## ⚡ 技能

### 脉冲光线

**类型**：被动技能（永久生效）

**效果**：菲尼击败敌人之后，敌人会对相邻的敌人放出脉冲光线，造成其（敌人）最大生命 20% 的伤害。

## 🎯 战术定位

连锁伤害型法师，击杀敌人后可触发额外伤害。适合清理密集敌群。
