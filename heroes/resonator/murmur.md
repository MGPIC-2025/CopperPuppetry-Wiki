# 絮絮 (Murmur)

## 📋 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 絮絮 |
| **英文名** | Murmur |
| **职业** | 共鸣者 (Resonator) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## 📖 背景故事

一位性格内向的织物艺术家创造的伴侣铜偶。"大猫"是由她收集的废弃铜线和皇家织毯的绒线构成的。大休眠时，她正把自己藏在未完成的编织作品里。重启后，她与"大猫"之间的共生联结变得更强，可以共享视觉和情感。

## 📊 基础属性

- **生命值**：100
- **攻击力**：10
- **速度**：3
- **防御力**：10
- **闪避**：10

## ⚡ 技能

### 大猫守护

**类型**：被动技能（永久生效）

**效果**：絮絮受到伤害时，大猫会抓伤敌人，造成固定的 20 点伤害。

## 🎯 战术定位

反伤辅助型英雄，受击时可以自动反击。适合作为半肉辅助。
