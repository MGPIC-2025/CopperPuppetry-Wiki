# 蜜拉 (Hive)

## <span class="steam-icon doc"></span> 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 蜜拉 |
| **英文名** | Hive |
| **职业** | 共鸣者 (Resonator) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## <span class="steam-icon doc"></span> 背景故事

皇家花园的授粉与护卫单位。蜂群能采集花蜜，也能用蜂刺驱逐害虫。大休眠似乎增强了蜂群的集群智能和攻击性。蜜拉的核心程序里依然保留着'保护花园'的最高指令，只是现在她的"花园"范围有点太广了。

## <span class="steam-icon doc"></span> 基础属性

- **生命值**：100
- **攻击力**：10
- **速度**：3
- **防御力**：10
- **闪避**：10

## <span class="steam-icon lightning"></span> 技能

### 蜂群守护

**类型**：被动技能（永久生效）

**效果**：蜜拉击败敌人之后，蜂群会守护蜜拉，蜂群会攻击蜜拉周围的敌人，造成 10 点伤害。

## <span class="steam-icon aim"></span> 战术定位

召唤型辅助，击杀敌人后可以召唤蜂群攻击。适合持续输出。
