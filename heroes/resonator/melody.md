# 啾啾 (Melody)

## 📋 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 啾啾 |
| **英文名** | Melody |
| **职业** | 共鸣者 (Resonator) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## 📖 背景故事

园林景观用的仿生鸣禽单元，旨在用悦耳的叫声营造宁静氛围。大休眠的能量脉冲超载了它的鸣叫模块，使其声音具备了实体能量。现在，它不仅能召唤机械鸟群，其高频鸣叫本身就能震碎脆弱的敌人。

## 📊 基础属性

- **生命值**：100
- **攻击力**：10
- **速度**：3
- **防御力**：10
- **闪避**：10

## ⚡ 技能

### 啾啾的歌

**类型**：被动技能（永久生效）

**效果**：啾啾击败敌人之后，与该敌人相邻的敌方单位（联通即可）全部都受到 5 点伤害。

## 🎯 战术定位

连锁伤害型辅助，击杀敌人后可传播伤害。适合清理密集小怪。
