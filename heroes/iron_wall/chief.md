# 铿铿 (Chief)

## 📋 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 铿铿 |
| **英文名** | Chief |
| **职业** | 铁壁 (Iron Wall) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## 📖 背景故事

一套微型城市管理单元的指挥核心。原本负责指挥工兵单元进行街道清扫和路灯维护。重启后，他的指挥协议阴差阳错地连接上了几个战斗型铜偶，于是他一丝不苟地将他们编成了"第一微型战术小队"，尽管队员们常常不听指挥。

## 📊 基础属性

- **生命值**：150
- **攻击力**：15
- **速度**：2
- **防御力**：20
- **闪避**：10

## ⚡ 技能

### 队长光辉

**类型**：被动技能（永久生效）

**效果**：铿铿身边 3x3 范围内的战士都可以获得 20% 的攻击力加成与 10% 的防御力加成，这一技能的计算总是会发生在所有被动技能之前，以免影响其他有属性换算的英雄角色的数值计算混乱。

## 🎯 战术定位

团队增益型坦克，为周围友军提供攻击和防御加成。是团队的核心支撑。
