# 大贝尔 (Bell)

## 📋 基础信息

| 属性 | 数值 |
|------|------|
| **中文名** | 大贝尔 |
| **英文名** | Bell |
| **职业** | 铁壁 (Iron Wall) |
| **稀有度** | ⭐⭐⭐⭐ (史诗) |

## 📖 背景故事

这不是一个仿生铜偶，而是一扇真正被赋予了意识的"辉煌大厅正门"。它对自己的抛光度和历史价值极为自豪。大休眠让它从合页上脱落，如今它依靠反重力核心行动。它时常念叨着想回到那个"所有显贵都要经过我才能入内"的时代。

## 📊 基础属性

- **生命值**：150
- **攻击力**：15
- **速度**：2
- **防御力**：20
- **闪避**：10

## ⚡ 技能

### 铜墙铁壁

**类型**：被动技能（永久生效）

**效果**：大贝尔的攻击力永远等于防御力的值（攻击力的成长与装备都无法影响攻击力的数值）。

## 🎯 战术定位

特殊机制坦克，攻击力和防御力绑定。堆防御装备即可同时提升攻击和防御能力。
